# Contributing

Thank you for considering contributing to this repository! 🎉
This project aims to provide a collection of useful Python scripts for automating common file and folder management tasks.

## How to Contribute

There are several ways you can contribute:

* 🐛 Report bugs
* 💡 Suggest new script ideas
* ✨ Add new automation scripts
* 🔧 Improve existing scripts

---

## Getting Started

1. **Fork the repository**
2. **Clone your fork**

```bash
git clone https://github.com/Saumyajeet-Varma/quickscripts.git
```

3. **Create a new branch**

```bash
git checkout -b feature/script-name
```

4. **Make your changes**

5. **Commit your changes**

```bash
git commit -m "Add: script for organizing downloads folder"
```

6. **Push the branch**

```bash
git push origin feature/script-name
```

7. **Open a Pull Request**

---

## Contribution Guidelines

Please follow these guidelines when contributing:

* Scripts should be written in **Python**
* Code should be **clear, readable, and well-structured**
* Add **comments where necessary**
* Each script should have a **descriptive filename**
* Avoid adding unnecessary dependencies

Example naming style:

```
delete_by_extension.py
move_by_extension.py
bulk_rename_files.py
```

---

## Script Requirements

Each new script should:

* Solve a **useful automation problem**
* Work on **multiple operating systems when possible**
* Include **basic input validation**
* Print clear messages for the user

---

## Pull Request Checklist

Before submitting a pull request:

* [ ] Code runs without errors
* [ ] Script follows the naming conventions
* [ ] Code is properly formatted
* [ ] Script solves a useful task
* [ ] No unnecessary files added

---

## Reporting Issues

If you find a bug or want to request a feature, please open an **Issue** and include:

* Description of the problem
* Steps to reproduce
* Expected behavior
* Screenshots (if applicable)

---

## Code of Conduct

Please be respectful and constructive when interacting with others.
All contributors are expected to maintain a friendly and collaborative environment.
