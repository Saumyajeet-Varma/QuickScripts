# Quick Scripts

A collection of useful Python scripts to automate common file and folder management tasks like deleting, moving, renaming, and organizing files.

| Script Name | Description |
|-------------|-------------|
| bulk_rename_files.py | Renames multiple files in a folder using a specified prefix or pattern. |
| clean_cache_files.py | Deletes cache and temporary files such as `.log`, `.tmp`, and `.cache`. |
| count_files_by_extension.py | Counts the number of files with a specific extension in a directory. |
| create_multiple_files.py | Creates multiple files automatically with a given prefix and extension. |
| create_multiple_folders.py | Generates multiple folders at once with a specified naming pattern. |
| delete_by_extension.py | Recursively deletes all files with a specified extension from a folder. |
| delete_duplicate_files.py | Detects and removes duplicate files using file hashing. |
| delete_empty_folders.py | Finds and removes all empty folders inside a directory. |
| find_large_files.py | Scans directories and lists files larger than a specified size. |
| move_by_extension.py | Moves files of a particular extension to a destination folder. |
