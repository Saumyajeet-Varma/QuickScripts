# QuickScripts

A collection of useful **Python scripts** to automate common file and folder management tasks.
These scripts help simplify repetitive operations such as deleting files, moving files, renaming files, and organizing directories.

---

## Scripts Overview

| Script Name                 | Description                                                             |
| --------------------------- | ----------------------------------------------------------------------- |
| bulk_rename_files.py        | Renames multiple files in a folder using a specified prefix or pattern. |
| clean_cache_files.py        | Deletes cache and temporary files such as `.log`, `.tmp`, and `.cache`. |
| count_files_by_extension.py | Counts files with a specific extension in a directory.                  |
| create_multiple_files.py    | Automatically creates multiple files with a given prefix and extension. |
| create_multiple_folders.py  | Generates multiple folders with a specified naming pattern.             |
| delete_by_extension.py      | Recursively deletes all files with a given extension.                   |
| delete_duplicate_files.py   | Detects and removes duplicate files using hashing.                      |
| delete_empty_folders.py     | Finds and removes empty directories.                                    |
| find_large_files.py         | Lists files larger than a specified size.                               |
| move_by_extension.py        | Moves files with a specific extension to another folder.                |

---

## Requirements

* Python 3.x
* No external dependencies required (uses only standard Python libraries)

---

## Usage

Run any script using Python:

```bash
python script_name.py
```

Example:

```bash
python delete_by_extension.py
```

Follow the prompts in the terminal to provide required inputs such as folder path or file extension.
